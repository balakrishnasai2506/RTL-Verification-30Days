# RTL-Verification-30Days

**30 Days · 90 Codes · Verilog + SystemVerilog + SVA**

A disciplined 30-day HDL coding challenge to strengthen **RTL Design** and **Design Verification** skills using Verilog, SystemVerilog, and SystemVerilog Assertions.

---

## 🎯 Objective

- Write **3 HDL codes per day** for 30 days
- Improve RTL design and verification thinking
- Build a consistent GitHub commit history
- Create a strong portfolio for VLSI/ASIC/Verification roles

---

## 🛠️ Topics Covered

- Verilog RTL design
- SystemVerilog language features
- SystemVerilog Assertions (SVA)
- Basic testbench concepts
- Writing clean, synthesizable, and testable code
